**HeroHome** is a modern web application that connects users with trusted local service
providers such as electricians, plumbers, and cleaners. Users can browse services, book
appointments, and leave ratings, while providers can manage their listings.

## Live link
https://herohome-8c1d7.web.app/

## Tech Stack

**Client:** React, Router, TailwindCSS, Firebase Auth.  
**Server:** Node Js, Express Js.  
**Npm packages:** Swiper.js, Motion, Hot toast, Sweet alert2, DaisyUi, Rechart, etc...


## Features

- Light/dark mode toggle
- Can booked services by other providers
- Add services
- Live statistics
- Cross platform
